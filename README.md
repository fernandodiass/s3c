💹 Sistema de Consulta de Cotações Cripto (S3C)
Sistema web responsivo para consulta histórica de preços de criptomoedas com conversão em tempo real para Real (BRL) e funcionalidades de exportação de relatórios.

🚀 Link do Projeto
Acesse a aplicação online: busca-cotacoes-cripto.vercel.app

✨ Funcionalidades
Busca Inteligente: Permite consultar preços históricos informando a sigla (BTC, ETH, etc.) ou o nome da moeda.

Histórico de Consultas: Os resultados são acumulados em uma tabela dinâmica durante a sessão.

Exportação XLS: Gera planilhas Excel dos dados consultados com um clique.

Exportação PDF: Gera um relatório profissional em PDF, otimizado para impressão (fundo branco e letras pretas).

Interface Dark Mode: Design moderno e elegante construído com o framework Bulma.

🛠️ Tecnologias Utilizadas
Frontend:
HTML5 & CSS3 (Bulma CSS Framework)

JavaScript (ES6+)

XLSX (SheetJS): Para geração de arquivos Excel.

jsPDF & AutoTable: Para criação de relatórios em PDF.

Backend:
Node.js: Operando em Serverless Functions na Vercel.

API CoinGecko: Fonte de dados para cotações históricas de criptoativos.

📋 Como usar
Digite o código da criptomoeda (ex: BTC ou SOL).

Selecione a Data e o Horário desejados.

Clique em Consultar.

O resultado aparecerá na tabela abaixo.

Utilize os botões Exportar XLS ou Exportar PDF para salvar seus dados localmente.
